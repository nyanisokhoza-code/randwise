🔔 Popups & Notifications — Global Controls
Settings saved to Supabase app_config table. Changes take effect within 30 seconds for all users — no app restart needed. Run the SQL below once to create all setting keys.
⚡ One-time SQL setup (run in Supabase SQL Editor)
INSERT INTO app_config (key, value) VALUES
  ('setting_daily_reminder', 'true'),
  ('setting_debt_intervention', 'true'),
  ('setting_debt_intervention_threshold', '60'),
  ('setting_share_nudge', 'true'),
  ('setting_share_nudge_frequency', 'weekly'),
  ('setting_whats_new_popup', 'true'),
  ('setting_about_screen', 'true'),
  ('setting_monthly_recap', 'true'),
  ('setting_upgrade_nudge', 'true'),
  ('setting_upgrade_nudge_text', 'Unlock the full power of MyRandWise — from R49/mo')
ON CONFLICT (key) DO NOTHING;
🔔 Daily 8pm Expense Reminder
Fires at 8pm if user hasn't logged any expenses today. Personalised with their name.
ON
To turn OFF: UPDATE app_config SET value='false' WHERE key='setting_daily_reminder';
To turn ON: UPDATE app_config SET value='true' WHERE key='setting_daily_reminder';
⚡ Debt Intervention Popup
Shows when user's debt exceeds the threshold % of income. Max once per week per user.
ON
Threshold options: 40% (early warning) · 60% (default) · 80% (critical only)
Change threshold: UPDATE app_config SET value='40' WHERE key='setting_debt_intervention_threshold';
Turn off: UPDATE app_config SET value='false' WHERE key='setting_debt_intervention';
🌱 Share & Earn Nudge
Fires on Saturdays. Reminds users to share the app. Frequency: weekly / biweekly / monthly.
ON
Set biweekly: UPDATE app_config SET value='biweekly' WHERE key='setting_share_nudge_frequency';
Set monthly: UPDATE app_config SET value='monthly' WHERE key='setting_share_nudge_frequency';
Turn off: UPDATE app_config SET value='false' WHERE key='setting_share_nudge';
✨ What's New Popup
Shows once per app version after an update. Never repeats for the same version.
ON
Turn off: UPDATE app_config SET value='false' WHERE key='setting_whats_new_popup';
📋 About MyRandWise Intro
Shows once to new users on first login. Explains what the app does. Can be reset for all users.
ON
Reset for all users (they'll see it once more): DELETE FROM app_config WHERE key='setting_about_seen_reset'; INSERT INTO app_config VALUES ('setting_about_seen_reset', NOW()::text);
Turn off completely: UPDATE app_config SET value='false' WHERE key='setting_about_screen';
🔒 Upgrade Nudge Message
Text shown on Pro feature lock cards. Edit to match your current promotion or season.
